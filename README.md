
# User Activity Pattern Detection using Apache Spark

## Project Overview
A Big Data Analytics mini project that detects user behavior
patterns and anomalies from e-commerce activity data using PySpark.

## Files
- `ecommerce_user_activity.csv` — Original dataset (103 records)
- `user_activity_spark.py`     — PySpark analysis code

## How to Run
1. Install Python 3 and Java 11
2. Run: pip install pyspark
3. Run: python user_activity_spark.py

## Results
- Peak activity hour: 13:00
- Most common action: view (65%)
- Anomalous users detected: 6 out of 20 (30%)
